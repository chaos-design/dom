# @chaos-design/dom-finder

## 使用

```tsx
import {
  getElementAttribute,
  getElementClassList,
  getElementId,
  getNthIndex,
} from '@chaos-design/dom-finder';

const id = getElementId(el);
const classList = getElementClassList(el);
const attrList = getElementAttribute(el);
const nthIndex = getNthIndex(el);
```
